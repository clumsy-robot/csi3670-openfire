Overview

Our Project

Our project is based on the Openfire RTC server and the Spark IM application, both provided by the company, ignite realtime. The Openfire server allows us to administer the Spark IM application, and the IM application connects to the Openfire server and allows communication with other users via instant message. Both are open source and compatible with Windows, Linux, and Mac OS X. We chose to install the Openfire software on Windows Server 2016. Service link: Openfire.

Technologies Used

  • Openfire RTC Server Software
  • Spark IM application
  • Windows Server 2016
  • PostgreSQL Database
  • GitHub
  • VMware vSpehere Web Client
  • OU Datacenter VMs

Server Environment

  • Openfire application is on the 'openfire' VM.
    • Pulls user information from AD. Stores server configuration properties and other info in the database.
  • PostgreSQL database is on the 'openfiredb' VM.
  • Domain controller is on the 'ad01' VM.

How to Connect to Service

In order to connect to the Openfire RTC server, you must open up the Spark IM application and use Windows authentication to log in to the client. Then you will be connected to the server.

SaaS

It classifies as software-as-a-service (SaaS), because it offers a central location (hosted by us) where clients connect in order to communicate with others via instant message.
